# Boat art

Drop the production asset here as **`loone-boat.png`** — the sketches load
`/assets/loone-boat.png` first.

Fallback chain when the PNG is absent:

1. `loone-boat.svg` (copied from Experimental-Visualizer)
2. A plain rectangle drawn between the two physics particles, so the
   buoyancy simulation stays verifiable with no art at all.
